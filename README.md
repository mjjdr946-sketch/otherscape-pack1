# Otherscape — Pack 1 (MST-SCA005)

10 personnages pré-tirés pour **Metro-Otherscape** (Mist Engine), en français.

## Contenu

| Metro Crew | Autre équipe |
|---|---|
| **Dokkaëbi** — farceur gobelin infiltrateur | **Genji** — onmyôji du Bureau de l'Onmyô |
| **Kerberoz** — designer de drones à trois têtes | **Horishi** — tatoueuse de yokais vivants |
| **Ocelot** — sicario fantomatique du cartel | **Shikigami** — esprit serviteur libre |
| **Shams** — lieur de djinns, chasseur de Bruit | **Unagi** — pilleuse bénie par la Princesse Anguille |
| **Valkyrie** — cobaye augmenté de Ragnarok Inc. | **Wilson** — mercenaire gaijin à la force d'oni |

Chaque personnage a :
- 5 themebooks complets (Loadout + 2 Mythos + 2 Logos)
- Tous les tags (power + weakness)
- Portrait 512×512
- Token circulaire 256×256

## Installation

### Via Foundry (recommandé)
1. **Modules → Install Module**
2. Colle l'URL du manifest :
   ```
   https://raw.githubusercontent.com/mjjdr946-sketch/otherscape-pack1/main/module.json
   ```
3. Active le module dans **Manage Modules**
4. Va dans l'onglet **Compendium** → glisse les acteurs dans ton monde

### Manuel (Docker/VPS)
```bash
cd /chemin/vers/foundry-data/Data/modules/
tar xzf otherscape-pack1-module-complete.tgz
docker restart <conteneur-foundry>
```

## Compatibilité
- Foundry VTT v14+
- Système city-of-mist v4.5.1+
